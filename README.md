# Machine Test Dashboard

A responsive, interactive dashboard built with **HTML**, **CSS (Bootstrap 5)**, and **JavaScript**. It includes metric cards, data tables with pagination, search functionality, and a collapsible sidebar for navigation.

---

## Features

- **Top Navigation Bar**
  - Sticky top nav with search input
  - User profile icon and greeting
  - Sidebar toggle (desktop & mobile)

- **Sidebar**
  - Collapsible sidebar
  - Active state highlighting
  - Smooth transition for expanding/collapsing

- **Metric Cards**
  - Interactive cards with hover effects
  - Customizable icons and colors
  - Responsive layout

- **Data Tables**
  - Search by name, title, email, location
  - Pagination with previous/next buttons
  - Rows per page selector
  - Sortable columns

- **Responsive Design**
  - Works on mobile, tablet, and desktop
  - Sidebar collapses on smaller screens
  - Metric cards adapt to screen width

- **Modals & Forms**
  - Small and large modal support
  - Clean form styling
